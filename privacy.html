Privacy Policy

Chinese pi is a Testnet application built on the Pi Network platform. This application is provided for testing and development purposes only.

Information We Collect

When you authenticate using the Pi Network SDK, we may receive your Pi user identifier (UID) and username. No other personal data is collected.

How We Use Information

The collected information is used solely to authenticate users and to enable Testnet payment functionality within the app.

Data Storage

No personal data is permanently stored on external servers. Any local data is stored only within the user's browser for session purposes.

Payments

All payments processed through this application are Testnet payments using Test-Pi. These transactions have no real-world value.

Third-Party Services

This app uses the Pi Network SDK and related Pi Network services. No other third-party services are used.

Contact

If you have questions regarding this Privacy Policy, please contact the app developer through the Pi Network developer platform.
